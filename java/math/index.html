    JAVA MATH CLASS :¶

    1. Floating-Point Precision Methods :¶

    A. Math.ceil(x) :¶

    Returns the value of x rounded down to its nearest integer.

    Similar to least-integer function in Math, i.e.,

    \mathbf{\lceil{x}\rceil = \text{LIF}}

    \text{Ex : }\lceil{8.2}\rceil = 9.0, \lceil{-2.3}\rceil = -2.0

    Math.ceil(8.2); // 9.0
Math.ceil(9.7); // 10.0
Math.ceil(-2.3); // -2.0
Math.ceil(-4.8); // -4.0

    B. Math.floor(x) :¶

    Returns the value of x rounded up to its nearest integer.

    Similar to greatest-integer function in Math, i.e.,

    \mathbf{\lfloor{x}\rfloor = \text{GIF}}

    \text{Ex : }\lfloor{8.2}\rfloor = 8.0, \lfloor{-2.3}\rfloor = -3.0

    Math.floor(8.2); // 8.0
Math.floor(9.7); // 9.0
Math.floor(-2.3); // -3.0
Math.floor(-4.8); // -5.0

    C. Math.round(x) :¶

    Returns the value of x rounded to its nearest integer.

    Math.round(8.2); // 8.0
Math.round(9.7); // 10.0
Math.round(-2.3); // -2.0
Math.round(-4.8); // -5.0

    Math Constants for \mathbf{\pi}** and e

    Math.PI // 3.14285714286


    Math.E // 2.7182818284

    2. Logarithmic and Exponential Methods :¶

    A. Math.log(x) :¶

    Returns the value of natural logarithm of x, i.e., \log_e{x} \text{ or } \ln{x}

    \text{Ex : }\log_{e}{e} = 1.0, \ln{1} = 0.0

    Math.log(Math.E); // 1.0
Math.log(0); // -Infinity
Math.log(1); // 0.0

    B. Math.log10(x) :¶

    Returns the value of \log_{10}{x}

    \text{Ex : }\log_{10}{10} = 1.0, \log_{10}{1} = 0.1

    Math.log10(10); // 1.0
Math.log10(100); // 10.0
Math.log10(1); // 0.1

    C. Math.exp(x) :¶

    Return the value of e raised to the power of x, i.e., e^x :

    \text{Ex : }e^0 = 0, e^1 = 2.7182

    Math.exp(0); // 1.0
Math.exp(1); // 2.7182818284
// The above is equal to Math.E

Math.exp(0.5); // 1.64872127

    3. Power and Root Methods :¶

    A. Math.sqrt(x) :¶

    Returns the value of square-root of x, i.e., \sqrt{x}

    \text{Ex : }\sqrt{9} = 3.0, \sqrt{2} = 1.414

    Math.sqrt(9); // 3.0
Math.sqrt(-36); // NaN
Math.sqrt(2); // 1.41421356237

    B. Math.cbrt(x) :¶

    Returns the value of cube-root of x, i.e., \sqrt[3]{x}

    \text{Ex : }\sqrt[3]{8} = 2.0, \sqrt[3]{-27} = -3.0

    Math.cbrt(8); // 2.0
Math.cbrt(-27); // -3.0
Math.cbrt(45); // 3.55689330449

    C. Math.pow(x, y) :¶

    Returns the value of x to the power of y, i.e., {x^y}

    \text{Ex : }2^3 = 8.0, 4^\frac{1}{2} = 2.0

    Math.pow(2, 3); // 8.0
Math.pow(3, 0); // 1.0
Math.pow(4, 0.5); // 2.0

    4. Sign-Related Methods :¶

    A. Math.abs(x) :¶

    Returns the absolute value of x.

    Similar to Modulus function in Math, i.e.,

    \mathbf{ \vert{x}\vert = \begin{cases} +x, & \text{if ${n > 0}$} \\ 0, & \text{if ${n = 0}$} \\ +x, & \text{if ${n < 0}$} \\ \end{cases} = \text{Modulus Function} }

    \text{Ex : }\vert50\vert = 50, \vert-50\vert = 50, \vert-50.5\vert = 50.5

    Math.abs(50); // 50
Math.abs(-50); // 50
Math.abs(-50.5); // 50.5

    B. Math.signum(x) :¶

    Similar to Signum function in Math, i.e.,

    \mathbf{ \{x\} = \begin{cases} 1.0, & \text{if ${x > 0}$} \\ 0, & \text{if ${x = 0}$} \\ -1.0, & \text{if ${x < 0}$} \\ \end{cases} = \text{Signum Function} }

    \text{Ex : }\{-50.5\} = 50.5

    Math.signum(2.3); // 1.0
Math.signum(-5.7); // -1.0
Math.signum(0); // 0.0

    5. Trigonometric Methods :¶

    A. Math.sin(x), Math.cos(x) :¶

    Returns the sine and cosine of x respectively, i.e., \sin x \text{ or } \cos x

    x should be passed in Radians.

    Math.sin(0); // 0.0
Math.sin(Math.PI / 2); // 1.0
Math.sin(-Math.PI / 2); // -1.0

Math.cos(0); // 1.0
Math.cos(Math.PI); // -1.0
Math.cos(-Math.PI); // -1.0

    B. Math.asin(x), Math.acos(x) :¶

    Returns the arc-sine and arc-cosine (or, sine and cosine inverses, respectively) of x respectively, i.e., \sin^{-1} x \text{ or } \cos^{-1} x

    x should be passed in Radians.

    Math.asin(0.0); // 0.0
Math.asin(1.0); // 1.57142857143
Math.asin(-1.0); // -1.57142857143

Math.acos(1.0); // 0.0
Math.acos(-1.0); // 3.14285714286
Math.acos(0.0); // 1.57142857143

    D. Math.hypot(x, y) :¶

    Returns the value of the hypotenuse of a right-angled triangle, assumed to be formed by x and y, i.e., \sqrt{x^2 + y^2}

    Same as doing this :

    Math.sqrt(Math.pow(x, 2) + Math.pow(y, 2))

    Math.hypot(3, 4); // 5.0
Math.hypot(5, 12); // 13.0
Math.hypot(5, 9); // 10.294630141
